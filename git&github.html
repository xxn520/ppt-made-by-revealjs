What‘s Git?

  • Git是目前世界上最先进的分布式版本控制系统。

What‘s 版本控制系统？

  • 记录和保存软件的每一次改动，并且能够在各个版本间切换
  • AND MORE!

传统的版本控制系统?

  • cvs
  • svn

它们与git的区别？

  • git是分布式的，意味着每台电脑都可以有一个完整的版本库
  • git不需要联网也可以进行工作
  • git有强大的分支管理

GitHub？？？

  • GitHub是一个免费的远程仓库
  • GitHub还是一个开源协作社区
  • 在GitHub上能够找到能想到的大多数库、组件
  • AND MORE！

Git基本使用

git init 初始化仓库

git status 查看仓库当前状态

git add 把文件添加到暂存区

git commit 把文件提交到本地仓库

git diff 查看文件被修改情况

git log 查看版本历史

git reset 退回到指定版本

git reflog 查看命令历史

git checkout -- file 撤销工作区中某个文件的修改

git reset HEAD file 把暂存区的修改撤销掉

git rm 删除文件

git checkout其实是用版本库里的版本替换工作区的版本

远程仓库

http、https或者ssh的方式

git remote add origin 添加远程仓库

git push -u origin master 将本地仓库的修改提交到远程仓库

git clone 最重要了
